# DGLife

DGIST 학생들의 편리한 학교 생활을 돕기 위한 어플리케이션.

## 주요 기능
- 학사 일정 확인
- 동아리 활동 관리
- 시설 예약
- 커뮤니티 게시판
- 알림 및 푸시 서비스

## 기술 스택

### 백엔드
- Framework: Spring Boot / FastAPI (선택)
- Database: MySQL (예시)

### 프론트엔드
| 플랫폼     | 추천 기술 스택                 | 개발환경(IDE)      |
| ------- | ------------------------ | -------------- |
| Android | Kotlin + Jetpack Compose | Android Studio |
| iOS     | Swift + SwiftUI          | Xcode          |
